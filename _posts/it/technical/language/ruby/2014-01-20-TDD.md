# 整体介绍

在敏捷开发的过程中, 

BDD要解决的一个关键问题是提升问题领域人员和解决方案领域人员之间的交流。
BDD的目的是为了创建并发展一个这两个领域之间相互理解的公共区域，创建一种公共语言，一种普及的语言（正如领域驱动设计中所定义的那样）。
