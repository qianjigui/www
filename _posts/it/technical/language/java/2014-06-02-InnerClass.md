# 一个例子

```java
public class B{
    private int tc;
    public B(int c){
        tc = c;
    }
    public void a(){
        final int cc = (new C(5)).c;
        new Thread() {
            @Override
                public void run() {
                    System.out.println(String.valueOf(cc));
                }
        }.start();
    }
    public static void main(String[] args){
        (new B(23)).a();
        int i =(new D()).d;
    }

    class C{
        int c;
        public C(int i){
            c=i;
        }
    }

    static class D{
        int d;
    }
}
```

# 静态内部类

不会添加对外部类的对象的引用.

# 普通内部类

与匿名内部类相同, 会在内部动态添加对父亲对象的引用.

# 匿名内部类
匿名内部类中, 通过编译器添加了成员变量保存final产生的数据.

```java
public class B{
    private int tc;
    public B(int c){
        tc = c;
    }
    public void a(){
        final int cc = tc;
        new Thread() {
            @Override
                public void run() {
                    System.out.println(String.valueOf(cc));
                }
        }.start();
    }
    public static void main(String[] args){
        (new B(23)).a();
    }
}
```
