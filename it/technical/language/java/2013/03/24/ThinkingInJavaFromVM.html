对于所有对象都通用的方法

  • 覆盖equals时请遵守通用约定
    • if(x!=NULL) x==x true
    • x and y are not NULL, x==y if and only if y==x
    • x,y,z NOT NULL, x==y y==z => x==z
    • if(x!=NULL) x==NULL return false
  • 覆盖equals时总要覆盖hashCode
    • Dalvik
      • MarkSweep GC
        • Object hashcode==object addr
      • Copying GC
        • Object hashcode is saved in the object
  • 始终要覆盖toString
    • For Debug
  • 谨慎地覆盖clone
  • 考虑实现Comparable接口
